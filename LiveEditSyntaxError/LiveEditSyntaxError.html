LiveEdit needs line numbers on syntax errors

  1. Open devtools, select Script Panel, unset Pause on all Exceptions,
  2. Click on the page. You should stop on line 15 debugger;
  3. Add a semicolon on line 9 after the comment, Control S to save

Expected: Console message: "SyntaxError: Unexpected token ';' line 9"

Actual: "Uncaught LiveEdit Failure: Failed to compile new version of script: SyntaxError: Unexpected token ;'"

This bug makes multi-point editing with LiveEdit hopeless. If you can't see which token is a problem, then you have to copy and paste your edits into another tool because you cannot save (see bug 123242)
